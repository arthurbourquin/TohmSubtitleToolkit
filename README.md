# Tohm Subtitle Toolkit

A single-page HTML tool for working with standard subtitle files (`.srt`).

---

## Features

- **Convert `.srt` to raster PNGs**  
  Generates images of each subtitle line that can be downloaded.

- **Convert `.srt` to After Effects `.jsx` script**  
  Creates a script that, when run in After Effects, generates timed text layers respecting line breaks.

- **Convert `.srt` to JSON object**  
  Produces a ready-to-use `.json` file from subtitle text and timings.

---

## How it works

All functionality is handled in-browser using **JavaScript**, with no dependencies.  
The HTML page manages file import, parsing, formatting, and download of results.

---

## Planned features

- **DCP XML for PNG subtitles**  
  Generate an XML file alongside PNGs for inclusion in Digital Cinema Packages.

- **Font customization**  
  Control color, style, font, stroke, and effects for rasterized subtitles.
